Quick Start
  1. Choose your use case (Tax Forms or Eng PPT Docs) from the dropdown in the header.
  2. Navigate to Step 1 – AI Search Health to verify the search index is healthy.
  3. Proceed to Step 2, select a sample prompt, and test the Foundry Agent directly.
  4. Continue through Steps 3 & 4 using the same prompt to verify APIM and Bot Service.
  5. In Step 5, build & download the agent package for Teams Developer Portal import.

Each step shows KPIs (response time, status, error details) in a master-detail layout.
