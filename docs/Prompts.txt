UI
===
Build a UI in Ionic, Angular, Typescript that has a separate layout to render in high quality for each device type (Web versus mobile versus tablet)
The UI will allow user to run multiple test scenarios sequentially following the data flow from AI Search to Foundry Agent to APIM to Bot service
In header, Choose a use case between Tax Forms and Eng PPT Docs at session level. Make default is tax forms
The input for sequential test case is to choose from one of sample prompts based on use case. The sample prompts are the ones from config json file under config folder
The following test cases wil run:
- Health check ai search and provide stats on ai search for this use case
- Test agent using selected prompt based on use case by calling foundry endpoint directly
- Test agent using selected prompt based on use case by calling APIM endpoint
- Test Bot service based on use case using same prompt
- Provide a page to package and download the agent to be imported in Teams dev portal and provide link to teams dev portal next to download
In each test case show detailed result with error messages in master - detailed format and KPI per test case.
The goal is to troubleshoot the data flow of this application from start to end.
In home page, explain the purpose of UI and steps to do.
