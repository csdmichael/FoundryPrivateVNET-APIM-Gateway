# Privacy Policy

## Overview

These Teams agent packages are part of the Foundry Private VNET APIM Gateway sample project. The apps submit user-entered prompts to an API endpoint fronted by Azure API Management, which then routes requests to a private Azure AI Foundry environment.

## Data Processed

The application may process:

- Prompt text entered by users in Microsoft Teams
- Grounding lookups against the configured Azure AI Search indexes
- Service telemetry and diagnostic logs generated by Azure resources used in the deployment

## How Data Is Used

Prompt data is used only to:

- Generate grounded responses for the active use case
- Retrieve relevant indexed content from the configured search source
- Support operational monitoring, debugging, and security review for the deployed sample

## Data Storage and Retention

This repository does not itself store user prompts. Data handling, retention, and logging depend on the Azure resources deployed for this project, including Azure API Management, Azure AI Foundry, Azure AI Search, and any connected monitoring resources.

## External Sharing

This sample is not intended to share prompt content with third-party services beyond the Azure services configured by the operator of the deployment.
